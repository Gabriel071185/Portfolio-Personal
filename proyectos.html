Django Suites Hotel

Página web dedicada a servicios de reserva, administración y gestión de habitaciones incluidos roomservice. Desarrollado con el framework Django y utiliza una base de datos relacional (PostgreSQL).

Geosanitas

Página web dedicada a localización de establecimientos sanitarios en la ciudad de Córdoba (Argentina), información sobre enfermedades y lesiones y formulario de contacto. Desarrollado con HTML, CSS, Javascript.

VW Phones

Página web que simula un e-commerce de smartphones con manejo de stock (ingreso-consulta-eliminación) con imágenes de modelos y características de cada uno. Desarollado con Python y el framework Flask, HTML, CSS, Javascript utilizando una base de datos realcional (MySQl).

SisGestra

Página web que emula un sistema de gestión de inventario y personal involucrado. Se destacó el diseño de una interfaz de usuario intuitiva, la implementación de un sistema de alertas para gestionar su correcto uso y la creación de reportes automatizados mediante envío de correo electronicos. Para su desarrollo, utilicé Python, Django y PostgreSQL.
